# 网文写作助手 - 产品需求文档

> 面向点众小说平台作者的专业写作工具

---

## 1. 产品概述

### 1.1 产品定位
为在**点众小说平台**投稿的作者提供专业的写作辅助工具，解决"新手启动困难"和"老作者写作累"两大核心痛点。

### 1.2 核心价值主张

| 用户群体 | 核心痛点 | 产品价值 |
|---------|---------|---------|
| 新手作者 | 不知道写什么、不会构建剧情 | 结构模板 + 实时诊断，快速入门 |
| 成熟作者 | 剧情管理困难、卡文、数据焦虑 | AI诊断 + 效率工具，持续输出 |

### 1.3 产品原则

**不做AI续写，只做AI诊断**
- ❌ 不替代作者创作（避免同质化）
- ✅ 辅助作者写得更好（结构诊断、质量把关）

## 2. 用户画像

### 2.1 目标用户

**平台**：点众小说（新媒体小说头部平台）
**用户层级**：
- 新手作者（0-1本书，未签约或刚签约）
- 进阶作者（1-3本书，有一定经验）
- 成熟作者（3本以上，稳定更新）

**写作类型**：中文网文为主，支持玄幻/都市/言情/悬疑等全品类

### 2.2 点众平台特性

| 特性 | 说明 | 产品应对 |
|-----|------|---------|
| 受众 | 下沉市场、30-50岁男性、碎片化阅读 | 节奏快、爽点密 |
| 付费模式 | 前几十章免费，关键卡点付费 | "卡点检测"功能 |
| 更新要求 | 日更4000-6000字是基本要求 | 字数统计、目标管理 |
| 热门题材 | 女婿文、神医、重生复仇、玄幻脑洞 | 热门题材模板库 |

## 4. 核心功能详解

### 4.1 写作工作台

#### 布局设计
- **左侧边栏**：章节导航（卷/章层级）+ 人物卡速览
- **中央区域**：富文本编辑器 + 字数统计
- **AI面板**：浮动按钮触发，按需展开

#### 编辑器功能
- 富文本编辑（支持常用格式）
- 自动保存（每30秒）
- 快捷键支持（Ctrl+S保存、Ctrl+F查找、Esc关闭弹窗）
- 专注模式（全屏无干扰）

### 4.2 AI诊断系统（核心差异化）

#### 设计理念
> AI不是替你写，而是帮你看

**诊断维度**：

| 维度 | 检测内容 | 输出形式 |
|-----|---------|---------|
| **结构完整性** | 起承转合进度 | 进度条可视化 |
| **情绪曲线** | 情绪起伏分析 | SVG曲线图 |
| **字数分布** | 开头/中间/结尾比例 | 堆叠条形图 |

#### 结构模板库

提供可复用的剧情结构框架：

1. **经典打脸结构（5步法）**
   - 受辱 → 隐忍 → 反击 → 震惊 → 余波

2. **悬念揭晓结构**
   - 铺垫 → 暗示 → 揭晓 → 影响

3. **情感爆发结构**
   - 压抑 → 触动 → 释放 → 余韵

**使用方式**：查看框架 → 理解结构 → 自己填充内容

#### 卡文检测
- 检测逻辑：5分钟无输入且已有内容
- 响应方式：温和询问（非强制打断）
- 提供选项：继续思考 / 查看模板建议

### 4.3 效率工具

#### 快捷输入
- 【】对话、（）心理、……省略、——分割线
- 🔥爽点、💔虐点标记

#### 智能取名
- 人名、地名、功法名、势力名
- 后续升级为"智能取名"（基于世界观设定）

## 5. 用户体验设计

### 5.1 配色方案

**莫兰迪暖色系**

浅色模式：
- 主背景：#F7F5F3（米白暖灰）
- 强调色：#C4A88C（驼色）
- 成功色：#9CAF94（灰绿）

深色模式：
- 主背景：#2D2A28（深暖灰）
- 强调色：#D4B896（浅驼色）

### 5.2 交互原则

1. **非侵入式AI**：AI面板默认收起，按需触发
2. **心流保护**：专注模式、卡文检测温和介入
3. **即时反馈**：字数实时统计、保存成功提示
4. **防丢稿保护**：切换章节检测未保存内容

### 5.3 新手引导

**首次使用流程**：
1. 欢迎页：选择作者类型（新手/进阶/成熟）
2. 题材选择：玄幻/都市/言情等
3. 模板推荐：基于选择推荐开头模板
4. 进入写作：预填示例内容，降低启动门槛

## 6. 技术方案

### 6.1 前端技术栈
- React + TypeScript
- Tailwind CSS（样式）
- Slate.js / Tiptap（富文本编辑器）
- Zustand（状态管理）

### 6.2 后端技术栈
- Node.js + Express / Fastify
- Prisma（ORM）
- PostgreSQL（主数据库）
- Redis（缓存）

### 6.3 部署
- Docker + Docker Compose
- 支持本地开发一键启动
