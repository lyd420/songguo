# 网文写作助手 v5

仿照点众文学创作者平台风格的网文写作工具。

## 与v4相比的改进

### UI设计
- **全新布局**：左侧导航栏 + 顶部Header + 卡片式主内容区
- **现代风格**：参考点众文学创作者平台，采用圆角、阴影、简洁的线条
- **色彩方案**：蓝色主色调(#1890ff)，搭配灰色背景和白色卡片
- **组件统一**：按钮、输入框、弹窗等组件风格统一

### 工作台页面
- **用户信息卡片**：顶部展示头像、笔名、累计创作字数
- **作品管理**：Tab切换短篇/长篇，书籍列表卡片式展示
- **码字日历**：显示每月写作情况，每日字数统计
- **公告活动**：右侧展示平台公告和活动信息

### 编辑器页面
- **三栏布局**：左侧目录/人物、中间编辑器、右侧标记面板
- **章节目录树**：卷-章节层级结构，可展开收起
- **人物卡片**：角色列表，支持详情面板和关系图谱
- **行标记**：点击行号添加todo/question/idea/issue标记

## 功能列表

### 核心功能
- ✅ 工作台：作品管理、码字日历、数据统计
- ✅ 章节目录：分卷管理、章节增删改
- ✅ 富文本编辑：字数统计、自动保存
- ✅ 人物卡系统：角色管理、关系图谱
- ✅ 行标记：四色标记、快速跳转
- ✅ 日更目标：目标设置、进度追踪
- ✅ AI助手：大纲/剧情/润色/扩写（界面预留）
- ✅ 主题切换：浅色/深色模式
- ✅ 专注模式：全屏写作

### 数据存储
- LocalStorage本地存储
- 章节内容、标记、人物数据持久化

## 技术栈
- 原生JavaScript (ES6 Modules)
- 纯CSS3 (CSS Variables、Flexbox、Grid)
- HTML5
- LocalStorage

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 无需安装任何依赖
3. 数据自动保存到浏览器本地存储
